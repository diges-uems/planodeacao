## Aviso de Segurança

A autenticação deste sistema é por senha simples via Google Apps Script, adequada para uso interno universitário (intranet/extranet controlada). Não é recomendada para dados sensíveis sem camada adicional de segurança. A URL do Apps Script está configurada diretamente no código em `src/lib/constants.ts` para facilitar a hospedagem estática no GitHub Pages. Se o sistema se tornar público além da comunidade acadêmica, é recomendável implementar uma camada de autenticação mais robusta.
